01 — Sobre mí

¿Quién soy?

Soy José Rubén Ramos Lomelí, desarrollador de software egresado con título de TSU en Desarrollo de Software en la Universidad Tecnológica de Aguascalientes. A lo largo de mi formación y proyectos personales he aplicado habilidades tanto de frontend como de backend, construyendo soluciones reales desde cero.

Me mueve la curiosidad, la mejora continua y el gusto por transformar ideas en productos funcionales. Cada proyecto que tomo es una oportunidad para aprender algo nuevo y entregar resultados que importen.

Actualmente busco oportunidades donde pueda seguir creciendo y aportar valor con código limpio, buenas prácticas y mucho compromiso.

Alta Rápida Z004

Aplicación nativa para Android diseñada para optimizar la gestión de inventarios en MyBusiness POS 20. Permite el alta rápida de productos directamente desde el dispositivo móvil, agilizando el trabajo en almacén o piso de ventas mediante su conexión segura por API.

Android Software de Gestión Productividad

MyBusinessBridge

Middleware para MyBusiness POS Delta. Servidor API REST construido con Node.js que conecta bases de datos SQL Server con aplicaciones móviles Android — consulta y gestión de inventarios en tiempo real, de forma segura y eficiente.

Node.js API REST SQL Server Android
